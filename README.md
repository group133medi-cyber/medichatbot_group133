# 🩺 MediCare AI – Medical Diagnostic Chatbot

MediCare AI is an AI-powered medical diagnostic chatbot developed using Python, Streamlit, Flask, and Hugging Face APIs.

The chatbot provides:
- Symptom-based medical guidance
- AI-generated health responses
- Emergency symptom detection
- Follow-up medical questions
- Conversation memory
- Interactive chatbot UI

# 🚀 Features

✅ AI-powered chatbot using Hugging Face API  
✅ Symptom detection using JSON medical dataset  
✅ Emergency severity identification  
✅ Follow-up questioning system  
✅ Streamlit interactive UI  
✅ Conversation memory support  
✅ Cloud deployment support  

# 🛠️ Technologies Used

- Python
- Streamlit
- Flask
- Hugging Face Inference API
- JSON
- HTML/CSS
